# Nextbike Map

Shows you all available Nextbikes in your area. Made with Vue.js, Vuetify and Leaflet (OSM) at #GPN19. See the [demo](https://bike.krmax44.de)

## Features

- easy and fast
- PWA
- included search
- GPS location
- navigate to available bike using a 3rd-party navigation app
